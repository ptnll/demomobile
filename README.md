## Hướng dẫn chạy app
1. Clone repo này về máy: `git clone https://github.com/ptnll/demomobile.git`
2. Cài đặt dependencies: `npm install` (hoặc `yarn install`)
3. Chạy app: `npm start` (hoặc `npx expo start` / `react-native run-android`)
