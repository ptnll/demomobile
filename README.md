# XÂY DỰNG ỨNG DỤNG DI ĐỘNG QUẢN LÝ THIẾT BỊ BẢO TRÌ TRƯỜNG HỌC

## Giới thiệu hệ thống
Hệ thống được phát triển nhằm hiện đại hóa quy trình quản lý tài sản và bảo trì trang thiết bị tại các cơ sở giáo dục. Thay vì sử dụng sổ sách truyền thống, ứng dụng cho phép:
* **Giáo viên:** Báo hỏng thiết bị.
* **Nhân viên kỹ thuật:** Tiếp nhận yêu cầu, cập nhật tiến độ và chi phí sửa chữa.
* **Quản trị viên (Admin):** Điều phối nhân sự, quản lý danh mục tài sản và theo dõi báo cáo thống kê trực quan.

Dự án sử dụng công nghệ **React Native** kết hợp hệ sinh thái **Expo**, đảm bảo ứng dụng vận hành mượt mà trên cả hai nền tảng Android và iOS.

## Hướng dẫn cài đặt và chạy project

### 1. Yêu cầu hệ thống
* Đã cài đặt **Node.js** (phiên bản LTS).
* Đã cài đặt **Git**.
* Điện thoại đã cài sẵn ứng dụng **Expo Go** (tải trên App Store hoặc CH Play).

### 2. Cài đặt
Mở Terminal và thực hiện các lệnh sau:
```bash
# Clone repo về máy
git clone [https://github.com/ptnll/demomobile.git](https://github.com/ptnll/demomobile.git)

# Di chuyển vào thư mục dự án
cd demomobile

# Cài đặt các thư viện phụ thuộc
npm install

# Khởi động Expo Server
npx expo start
